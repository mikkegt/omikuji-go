# omikuji-go

## How to start

```
git@github.com:mikkegt/omikuji-go.git
cd omikuji
vgrun devserver.go   
```

## Note

### Vugu setup

```
go install github.com/vugu/vgrun@latest
vgrun -install-tools

go get github.com/vugu/vugu
go get github.com/vugu/vugu/cmd/vugufmt
```

```
vgrun -new-from-example=simple myProject
```
